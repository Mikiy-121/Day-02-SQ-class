# AGENTS

This repository is a small Python practice workspace containing a single file:
- `Tip_split.py`

## Agent guidance

- Only provide code changes, suggestions, or explanations when explicitly requested by the user.
- Do not offer unsolicited or random suggestions while the user is typing code.
- Keep responses concise and directly relevant to the user request.
- Ask clarifying questions when the user request is ambiguous.

## Project notes

- There is no build/test system present in this workspace.
- The repo currently appears to be focused on simple Python exercises.
